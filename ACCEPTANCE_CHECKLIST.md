## 📋 需求核对表

### 排序功能 ✅

- [x] **需求**: 表格的排序功能
- [x] **升序/降序/原序切换**
  - [x] 第1次点击 → 升序
  - [x] 第2次点击 → 降序
  - [x] 第3次点击 → 原序
  - [x] 继续点击循环切换
- [x] **排序后自动刷新表格**
- [x] **单列排序（非多列）**
- [x] **智能排序**
  - [x] 数值列按数值排序
  - [x] 文本列按字母排序
  - [x] NULL 值排在最前
- [x] **第0列（复选框）不支持排序**

### 列选中功能 ✅

- [x] **需求**: 表格中可以选中一列
- [x] **点击表头选中整列**
- [x] **视觉反馈（高亮背景）**
  - [x] 浅蓝色背景 (RGB: 200, 220, 240)
  - [x] 整列高亮（包括所有行）
- [x] **支持多列选中**
  - [x] Ctrl+点击添加/移除列
  - [x] Shift+点击范围选中
- [x] **选中列显示在日志**
  - [x] 格式: 📌 选中列：列1, 列2, ...

### 实现方式 ✅

- [x] **继续用 QStandardItemModel**（实际是 QTableWidget）
- [x] **无需改动现有代码逻辑**（只新增一个模块 + 少量改动）
