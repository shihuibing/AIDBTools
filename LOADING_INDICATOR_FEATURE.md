# 加载状态指示器功能说明

## 功能概述

已成功为AI对话窗口添加了加载状态指示器，提升用户在等待AI响应时的体验。

## 实现的功能

### 1. 三点跳动动画气泡
- 显示"AI 正在思考中..."文本
- 三个点依次上下跳动，提供视觉反馈
- 自动更新省略号数量（. / .. / ... / ....）

### 2. 发送按钮状态变化
- 发送后按钮文本变为"等待中"
- 按钮透明度降低，表示禁用状态
- 防止用户重复点击发送

### 3. 请求耗时统计
- AI回复完成后自动显示请求耗时
- 格式：`✅ 请求完成（耗时 X.Xs）`
- 帮助用户了解响应速度

### 4. 完整的生命周期管理
- 发送消息时自动显示加载指示器
- 收到回复后自动隐藏并清理资源
- 支持手动停止时正确清理

## 修改的文件

### `ui/ai_chat_window.py`

#### 新增类
- `_LoadingBubble` - 加载状态指示器组件（约100行代码）

#### 修改的方法
- `AIChatWidget.__init__()` - 添加加载指示器相关属性
- `AIChatWidget._set_running()` - 集成按钮状态和耗时统计
- `AIChatWidget._show_loading_indicator()` - 显示加载指示器
- `AIChatWidget._hide_loading_indicator()` - 隐藏并清理加载指示器
- `AIChatWidget._call_ai()` - 普通对话模式集成
- `AIChatWidget._on_ai_reply()` - 普通对话回复处理
- `AIChatWidget._call_agent()` - Agent模式集成
- `AIChatWidget._on_agent_done()` - Agent完成处理
- `AIChatWidget._on_stop()` - 停止时清理
- `AIChatWidget._force_stop_check()` - 强制停止时清理

## 使用方式

加载指示器是**完全自动化**的，无需额外配置：

1. **普通对话模式**：
   - 用户输入消息并点击"发送"
   - 立即显示加载气泡和按钮状态变化
   - AI回复后自动隐藏，显示耗时统计

2. **Agent模式**：
   - 同样的自动显示/隐藏机制
   - 在Agent执行多个步骤期间持续显示
   - 所有步骤完成后隐藏

3. **手动停止**：
   - 点击"停止"按钮会立即隐藏加载指示器
   - 5秒超时强制停止也会清理

## 技术细节

### 动画实现
使用 `QTimer` 每150ms更新一次三个点的位置，通过 `margin-top` CSS属性实现上下跳动效果。相比 `QVariantAnimation`，这种方式更轻量且兼容性更好。

### 资源管理
- 加载指示器在隐藏时会调用 `deleteLater()` 确保内存释放
- 动画定时器会被正确停止，避免内存泄漏
- 支持多次显示/隐藏而不会创建多个实例

### 主题适配
加载指示器自动跟随当前主题（亮色/暗色）：
- 使用 `_tokens()` 获取主题配色
- 气泡背景、边框、文字颜色自动适配
- 头像颜色与AI回复保持一致

- 查看加载动画效果
- 点击"停止动画"按钮测试清理
- 点击"新建加载指示器"测试多次创建

### 集成测试
```bash
python main.py
```
然后在应用中：

3. 观察加载指示器显示
4. 等待AI回复，观察自动隐藏和耗时显示

## 性能影响

- **CPU**: 动画刷新率 ~6fps，几乎无影响
- **网络**: 无额外网络请求
