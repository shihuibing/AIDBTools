# AIDBTools 版本变更日志

## v1.0.0 (2026-03-31)

> 首个正式发布版本

### 新功能
- 支持 MySQL / PostgreSQL / SQL Server / Oracle 数据库连接与管理
- 支持国产数据库：达梦（DM）、人大金仓（KingbaseES）、高斯（GaussDB/openGauss）、OceanBase、PolarDB、TDSQL、GBase、TiDB、神通
- 支持星环科技（Transwarp）ArgoDB / Inceptor（JDBC + ODBC 双模式，自动探测驱动）
- AI 自然语言生成 SQL / AI 优化 SQL
- AI 对话助手（内嵌右侧面板）
- 大模型配置（多模型支持）
- Skill 管理（技能导入/应用）
- 数据导出 / 导入（CSV、Excel、JSON 等格式）
- 数据同步（表/库级别跨连接同步）
- 定时任务管理（支持多种触发类型）
- 数据库备份与恢复
- Navicat .ncx 连接文件批量导入（含密码填写预览）
- 主题切换（亮色 / 暗色 / 跟随系统）
- 结果表格分页浏览（可配置每页行数）
- MCPMA 支持

### 平台支持
| 版本 | 目录 | 说明 |
|------|------|------|
| Windows | `release/windows/v1.0.0/` | `AIDBTools_v1.0.0_windows.exe` |
| Linux | `release/linux/v1.0.0/` | `AIDBTools_v1.0.0_linux_x86_64` 或 `_aarch64` |
| 国产系统 | 源码 + `install_kylin.sh` | 银河麒麟V10 / 统信UOS / Deepin / openEuler |
