## 🏗️ 技术架构

### 核心类设计

**SortableTableHeader(QHeaderView)**
- 管理表头排序状态
- 实现排序逻辑（升/降/原）
- 信号发送排序改变事件

**SelectableTableWidget(QTableWidget)**
- 扩展 QTableWidget
- 集成排序和列选中功能
- 提供高亮显示管理

### 排序算法
```python
1. 收集表格数据 → 
2. 尝试数值排序 → 
3. 失败转字符串排序 → 
4. 用 blockSignals 防止信号干扰 → 
5. 重新设置 item
```

### 列选中实现
```python
1. 监听表头点击事件 →
2. 根据修饰键判断操作 →
3. 更新选中列集合 →
4. 重绘高亮 →
5. 发出选中改变信号
```
