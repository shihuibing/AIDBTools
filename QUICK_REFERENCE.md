# 🚀 快速参考卡 - 表格排序和列选中

## 操作速查表

### 🔤 排序操作

| 操作 | 快捷方式 | 效果 |
|------|---------|------|
| **开启排序** | 点击表头 | 按该列升序排列 |
| **切换降序** | 再点一次表头 | 改为降序排列 |
| **取消排序** | 再点一次表头 | 恢复原始顺序 |
| **切换列** | 点击不同的表头 | 按新列升序排列 |

### 🎯 选列操作

| 操作 | 快捷方式 | 效果 |
|------|---------|------|
| **单列选中** | 点击表头 | 选中整列（蓝色高亮） |
| **添加到选中** | Ctrl + 点击表头 | 同时选中多列 |
| **移除选中** | Ctrl + 点击已选列 | 取消该列选中 |
| **范围选中** | Shift + 点击表头 | 选中一个范围的列 |
| **查看选中列** | 查看日志 | 📌 选中列：... |

## 排序示例

**表格数据** (默认顺序):
```
名字   年龄
Alice  25
Bob    30
Carol  28
```

**点击「年龄」表头**:
```
名字   年龄    ← 升序
Carol  28
Alice  25
Bob    30
```

**再点一次「年龄」表头**:
```
名字   年龄    ← 降序
Bob    30
Carol  28
Alice  25
```

**再点一次「年龄」表头**:
```
名字   年龄    ← 原序（恢复）
Alice  25
Bob    30
Carol  28
```

无高亮

选中「名字」列 (蓝色高亮)

## ⚡ 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| **Click** (单击) | 排序 + 单列选中 |
| **Ctrl + Click** | 多列选中/取消 |
| **Shift + Click** | 范围选中 |
| **Tab** | 导出选中行 |

## 💡 常用场景

### 场景 1: 按工资从高到低排序
1. 点击「工资」表头 ← 升序
2. 再点一次「工资」表头 ← 降序 ✓

### 场景 2: 同时查看多个列的数据
1. 点击「部门」表头 ← 选中
2. Ctrl + 点击「职位」表头 ← 再选
3. Ctrl + 点击「工资」表头 ← 继续选
4. 查看日志确认选中了哪些列

### 场景 3: 按年龄范围查看
1. 点击「年龄」表头 ← 升序排列
2. 然后手动查看年龄段数据

### 场景 4: 选中一行数据的多列进行复制
1. Shift + 点击某列 ← 范围选中
2. 右键复制（后续功能）

## 🚫 注意事项

| ⚠️ | 说明 |
|----|------|
| **第0列不排序** | 复选框列无法排序，点击无效 |
| **翻页重置排序** | 翻页后排序状态会自动重置 |
| **NULL 值排最前** | 升序时排在最前，降序时排在最后 |
| **数值智能识别** | 系统自动判断列是数值还是文本 |

## 📊 状态指示

**排序状态** (每列独立):
- `升序` → 第1次点击表头
- `降序` → 第2次点击表头  
- `原序` → 第3次点击表头

**选中状态** (可多个列):
- `未选` → 白色背景
- `已选` → 蓝色背景 (RGB:200,220,240)

## ❓ 快速问题解答

**Q: 怎么同时按两列排序?**
A: 当前不支持，但可在新增列后点击表头重新排序。

**Q: 排序后怎么恢复原顺序?**
A: 点击表头第3次，或者翻页后会自动重置。

**Q: 能否保存排序设置?**
A: 暂不支持，翻页后会重置（未来改进方向）。

**Q: 多少行数据开始变慢?**
A: 100K 行以上建议使用分页或 SQL 级排序。
