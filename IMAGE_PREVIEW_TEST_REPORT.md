## Usage

### Upload Images via Button
1. Click the upload button (paperclip icon) in the input area
2. Select image files from the file dialog
3. Thumbnails appear above the input box
4. Click any thumbnail to preview full size

### Drag and Drop
1. Drag image files from file explorer
2. Drop onto the input text area
3. Thumbnails automatically appear
4. Files are added as @image:filename mentions

### Preview Images
1. Click any thumbnail in the input area
2. Full-size preview opens in modal dialog
3. Click "Close" to dismiss
